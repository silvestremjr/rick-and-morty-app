# Rick and Morty App

App developed using Rick and Morty Graphql in Vue.js

### Installing
Install dependencies

```
yarn install
```

### Run
Run local serve

```
yarn serve
```

## Running the tests

All components are covered with unit tests.
To run component tests:

```
yarn jest
```

## Storybook

To run component storybook:

```
yarn storybook
```
